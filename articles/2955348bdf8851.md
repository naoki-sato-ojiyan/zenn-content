## はじめに

- macOS Sonoma

## やること

1. Homebrewのインストール
2. Pythonのインストール
3. Gitのインストール
4. GitHubのSSH設定

## 1. Homebrewとは？

Macのアプリやツールをコマンドでインストールできるパッケージマネージャーです。
開発者はほぼ全員使っています。

### インストール

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

インストール後はPATHの設定が必要です。

```bash
echo >> ~/.zprofile
echo 'eval "$(/opt/homebrew/bin/brew shellenv zsh)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv zsh)"
```

確認👇

```bash
brew --version
```

## 2. Pythonのインストール

Mac標準のPythonではなくHomebrew版をインストールします。

```bash
brew install python@3.13
```

PATHを通します。

```bash
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zprofile
source ~/.zprofile
```

確認👇

```bash
which python3
# /opt/homebrew/bin/python3 と表示されればOK
python3 --version
```

## 3. Gitのインストール

```bash
brew install git
```

GitHubの情報を登録します。

```bash
git config --global user.name "GitHubのユーザー名"
git config --global user.email "GitHubのメールアドレス"
git config --global init.defaultBranch main
```

## 4. GitHubのSSH設定

SSH鍵を生成します。

```bash
ssh-keygen -t ed25519 -C "GitHubのメールアドレス"
```

公開鍵をコピーします。

```bash
pbcopy < ~/.ssh/id_ed25519.pub
```

GitHubの `Settings` → `SSH and GPG keys` → `New SSH key` に貼り付けます。

接続テスト👇

```bash
ssh -T git@github.com
# Hi ユーザー名! You've successfully authenticated と表示されればOK
```
