## 1. 変数・定数（let・const）

```javascript
// const → 変更不可（定数）
const playerName = 'ゆうしゃ';

// let → 変更可能（変数）
let playerHp = 30;
let playerLevel = 1;
```

### constとletの使い分け

```
const → 名前など変わらないものに使う
let   → HPなど変わるものに使う
```

:::message
現場の慣習：基本的に`const`を使い、変更が必要なものだけ`let`を使う。`var`は使わない。
:::

## 2. 条件分岐（if文）

```javascript
let playerHp = 30;

// 敵から大ダメージを受ける
playerHp = playerHp - 35;

if (playerHp <= 0) {
    document.getElementById('message').textContent = 'ゲームオーバー！';
} else {
    document.getElementById('message').textContent = 'HP:' + playerHp;
}
```

## 3. 関数（function）

同じ処理をまとめて何度でも呼び出せる仕組みです。

```javascript
let playerHp = 30;

// ダメージを受ける関数
function takeDamage(damage) {
    playerHp = playerHp - damage;

    if (playerHp <= 0) {
        document.getElementById('message').textContent = 'ゲームオーバー！';
    } else {
        document.getElementById('message').textContent = 'HP:' + playerHp;
    }
}

// 関数を呼び出す
takeDamage(10);  // HP:20
takeDamage(15);  // HP:5
```

```
関数の定義　→　レシピを作る
関数の呼び出し→　レシピ通りに料理する（何回でもできる）
```

## 4. 配列・オブジェクト

```javascript
// 配列：複数のデータをまとめて管理
const questions = [
    '8 × 7 = ?',
    '9 × 9 = ?',
    '144 ÷ 12 = ?',
];

// 配列の中身を取り出す（0番目から始まる）
console.log(questions[0]);  // 8 × 7 = ?
console.log(questions[1]);  // 9 × 9 = ?

// オブジェクト：関連するデータをまとめて管理
const player = {
    name: 'ゆうしゃ',
    hp: 30,
    level: 1,
    attack: 10,
};

console.log(player.name);    // ゆうしゃ
console.log(player.hp);      // 30
```

### オブジェクトの配列（まなびクエストで使う形式）

```javascript
const questions = [
    { q: '8 × 7 = ?', a: 56, choices: [54, 56, 63, 48] },
    { q: '9 × 9 = ?', a: 81, choices: [72, 79, 81, 83] },
    { q: '144 ÷ 12 = ?', a: 12, choices: [11, 13, 12, 14] },
];
```

## 5. 繰り返し（for・forEach）

```javascript
// for文
for (let i = 0; i < questions.length; i++) {
    console.log(i + '問目：' + questions[i].q + ' 答え：' + questions[i].a);
}

// forEach文（よりシンプルな書き方）
questions.forEach(function(question, index) {
    console.log(index + '問目：' + question.q);
});
```

### for文とforEachの違い

```
for文　　→　インデックス（番号）を使いたいとき
forEach　→　シンプルに全要素を処理したいとき
```

## 6. DOM操作

HTMLの要素をJavaScriptで操作することを**DOM操作**といいます。

```javascript
// テキストを変更
document.getElementById('message').textContent = '正解！';

// HTMLごと変更
document.getElementById('message').innerHTML = '<strong>正解！</strong>';

// 中身を全消し
document.getElementById('buttons').innerHTML = '';
```

### textContentとinnerHTMLの違い

```
textContent → テキストだけ操作（安全）
innerHTML   → HTMLタグごと操作できる
```

## 8. 動的HTML生成（createElement・appendChild）

問題が変わるたびにボタンを作り直す仕組みです。

```javascript
// ① 前のボタンを全消し
const buttonsDiv = document.getElementById('buttons');
buttonsDiv.innerHTML = '';

// ② ボタンを動的に作る
choices.forEach(function(choice) {
    const btn = document.createElement('button');  // <button>を作る
    btn.textContent = choice;                      // テキストをセット
    btn.onclick = function() { checkAnswer(choice); }; // クリック時の処理
    buttonsDiv.appendChild(btn);                   // divに追加
});
```

```
createElement → 要素を作る（まだ画面に出ない）
appendChild  → 要素を画面に追加する
innerHTML='' → 既存の要素を全部消す（リセット）
```

## 9. 全部まとめた問題クイズ

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>まなびクエスト</title>
</head>
<body>
    <h1>まなびクエスト</h1>
    <p id="question"></p>
    <div id="buttons"></div>
    <p id="message"></p>

    <script>
        const questions = [
            { q: '8 × 7 = ?', a: 56, choices: [54, 56, 63, 48] },
            { q: '9 × 9 = ?', a: 81, choices: [72, 79, 81, 83] },
            { q: '144 ÷ 12 = ?', a: 12, choices: [11, 13, 12, 14] },
        ];

        let currentIndex = 0;

        function showQuestion() {
            const current = questions[currentIndex];
            document.getElementById('question').textContent = current.q;

            const buttonsDiv = document.getElementById('buttons');
            buttonsDiv.innerHTML = '';
            current.choices.forEach(function(choice) {
                const btn = document.createElement('button');
                btn.textContent = choice;
                btn.onclick = function() { checkAnswer(choice); };
                buttonsDiv.appendChild(btn);
            });
        }

        function checkAnswer(selected) {
            const correct = questions[currentIndex].a;
            if (selected === correct) {
                document.getElementById('message').textContent = '正解！';
                currentIndex++;
                if (currentIndex < questions.length) {
                    showQuestion();
                } else {
                    document.getElementById('message').textContent = '全問正解！🎉';
                }
            } else {
                document.getElementById('message').textContent = 'はずれ！';
            }
        }

        showQuestion();
    </script>
</body>
</html>
```

## まとめ

| 概念 | 使いどころ |
|---|---|
| `const` / `let` | 変数・定数の宣言 |
| `if / else` | 条件分岐 |
| `function` | 処理をまとめる |
| 配列・オブジェクト | データをまとめて管理 |
| `for` / `forEach` | 繰り返し処理 |
| `textContent` | テキストの変更 |
| `innerHTML` | HTMLごと変更・削除 |
| `createElement` | 要素を動的に作る |
| `appendChild` | 要素を画面に追加 |
| `onclick` | クリックイベント |
